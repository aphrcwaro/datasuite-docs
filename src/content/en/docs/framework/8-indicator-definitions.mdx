## Indicator definitions

For cross-country standardization of the analysis, this section describes the indicators/variables extracted from the countries' HMIS, that are required for the Datasuite analysis.  
The data is grouped into three main categories:

- **Admin Data**
- **Population Data**
- **Service data**

---

### Admin Data

The following variables describe health system infrastructure and workforce.

| Variable Name | Description |
|--------------|------------|
| `District_name` | District |
| `First_admin_level` | Region |
| `Country` | Your country name |
| `Number_hospitals` | Total number of hospitals (based on country definition) |
| `Number_hcenters` | Total number of health centres (non-hospitals; based on country definition) |
| `Total_number_health_facilities` | Total number of health facilities (all types/categories) |
| `Number_pfacilities_profit` | Total number of health facilities that are private-for-profit |
| `Number_pfacilities_nonprofit` | Total number of health facilities that are private-non-profit |
| `Total_physicians` | Total number of physicians / Medical officers |
| `Total_nurses_midwives` | Total number of nurses / midwives |
| `Total_clinical_officers` | Total number of clinical officers / Assistant medical officers |
| `Total_nonclinique_physicians` | Total number of non-clinician physicians |
| `Number_hospital_beds` | Number of hospital beds |

---

### Population Data

The following variables describe demographic and population estimates.

| Variable Name | Description |
|--------------|------------|
| `District_name` | District name |
| `Year` | Year |
| `Pop_growth_rate` | Annual population growth rate |
| `Total_Population` | Total population |
| `Population_under_5years` | Population under 5 years |
| `Population_under_1year` | Population under 1 year |
| `Live_births` | Projected total live births (estimated from census) |
| `Total_births` | Projected total births (estimated from census) |
| `Women_15_49_years` | Total women aged 15–49 years |
